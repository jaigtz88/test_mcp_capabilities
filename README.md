# Angular Toast Notifications Demo

A sample Angular application demonstrating the usage of the `angular-toast-notifications` library with JSON-based configuration.

## Features

- Toast notification service with multiple types (success, error, warning, info)
- JSON-based configuration with environment-specific overrides
- Custom toast positions and durations
- Progress bar and pause on hover support
- Demo components showing various toast implementations

Refer to the [documentation](./doc/README.md) for detailed setup and usage instructions.
